# ARふくい

3Dスキャンされた3Dオープンデータを収集するプロジェクトです。

## デモ
https://code4fukui.github.io/ar-fukui/

## 特徴
- 「花輪 (祝 Hana道場 6周年 by IchigoJam)」の3Dモデルを提供

## データ / API
3Dデータは[Scaniverse](https://apps.apple.com/jp/app/scaniverse-lidar-3d-scanner/id1541433223)アプリを使用して作成されました。
