# AR Fukui

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A project to collect 3D open data scanned in 3D.

## Demo
https://code4fukui.github.io/ar-fukui/

## Features
- Provides a 3D model of "Hanawa (Congratulations Hana Dojo 6th Anniversary by IchigoJam)"

## Data / API
The 3D data was created using the [Scaniverse](https://apps.apple.com/jp/app/scaniverse-lidar-3d-scanner/id1541433223) app.
